## [POST] /api/v1/catalog/save-result - Экспортировать результаты в SQL скрипт

> [!TIP]
> API предназначен для выгрузки всех данных БД в SQL-скрипты, где каждый скрипт соответствует своей таблице.  
> Также дополнительно создаётся скрипт создания внешних ключей таблиц.  
> Все создаваемые в результате вызова данного API скрипты помещаются в директорию `/sql`, лежащую в корне сервиса
> _translator_, и предназначены для переноса в аналогичную директорию сервиса _mib-parser_, который управляет схемой БД.

### Возможные коды ошибок

400: Bad Request - Ошибка валидации конфигурационных данных для экспорта
500: Internal Server Error - Внутренняя ошибка сервера при записи файлов на диск или дампе таблиц СУБД
